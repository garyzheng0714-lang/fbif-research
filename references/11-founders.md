# 模块十一：创始人与关键人物深度背景

## ⚠️ 字数红线：≥5,000字。不到5,000字 = 不达标，必须补充搜索。

## 模块定位

**本模块只做一件事：深入了解推动品牌发展的关键人物的个人背景。**

人，始终是最重要的。每一个品牌背后，都有具体的人在推动。了解这些人的背景、经历、性格，是理解品牌决策逻辑的根基。一个设计专业出身的产品经理会做出和MBA出身的经理完全不同的决策。一个在行业里摸爬滚打20年的老兵和一个跨界进入的新人，面对同一个市场机会的反应也完全不同。

### 与其他模块的分工

| 内容 | 归属模块 | 本模块处理方式 |
|------|----------|---------------|
| 品牌时间线上的事件 | 模块二（品牌历史） | 不重复，只引用与人物相关的节点 |
| 创始人的公开采访内容 | 模块八（深度访谈） | 不重复正文，但可以引用关键履历信息 |
| 管理层变动事实 | 模块二（品牌历史） | 不重复，聚焦人物本身的背景 |

**原则：模块二记录"公司发生了什么"，本模块记录"这些人是谁、从哪来"。**

## 调研对象

### 谁是"关键人物"？

按优先级排序：

1. **创始人**（最重要，必查）
2. **联合创始人**（如有，必查）
3. **现任CEO**（如非创始人，必查）
4. **关键产品/项目的负责人**（如有公开信息）
5. **对品牌转折起到决定性作用的人**（如有公开信息）

不需要把整个管理层都查一遍。聚焦在2-5个真正推动了事物发展的人。

## ⚠️ 核心原则：原文一字不漏

**本模块要求对所有涉及关键人物个人经历的原文进行"地毯式"保留。**

找到一篇采访、一段人物介绍、一个播客片段中提到了创始人的童年、成长、喜好、重大经历——就必须把相关段落的原文完整抓下来写入 `sources/src-xxxx.md`，一个字都不能漏。不要概括、不要缩写、不要"提取关键信息"。先保留原文全文，再做中文翻译，再整理到模块文件中。

**为什么？** 因为人物的个人经历是深度文章中最有感染力的素材。一句创始人回忆童年的原话，一段关于他成长环境的描述，往往比十页财务数据更能打动读者。这些细节一旦在调研阶段被概括掉、省略掉，写作阶段就再也找不回来了。

## 调研内容（按人物逐个整理）

对每一个关键人物，按以下结构整理。**前六项是重点搜索项，必须穷尽所有可用来源：**

### 一、基本信息（必查）

- 全名（原文 + 中文，遵循命名铁律）
- 当前职位
- 国籍/所在地
- 出生年份（如公开）
- 家乡/出生地（如公开）

### 二、童年与早期经历（重点搜索）⭐ 新增重点

**这是最容易被忽略、却最有写作价值的部分。**

必须搜索的内容：
- 在哪里长大？什么样的家庭环境？（城市/农村、富裕/普通、父母职业）
- 小时候有什么特别的经历或记忆？（搬家、移民、家庭变故、文化冲击等）
- 童年时期的兴趣和爱好是什么？
- 有没有早期展现出创业特质的故事？（比如小时候卖柠檬水、帮家里做生意）
- 家庭对食物/饮食的态度如何？（对食品饮料品牌创始人尤为重要）
- 有没有影响深远的童年事件？（如某次旅行、某本书、某个人的影响）

**搜索关键词**：
1. `{人名} childhood / grew up / growing up`
2. `{人名} early life / young / youth`
3. `{人名} parents / family / hometown`
4. `{人名} 小时候 / 童年 / 成长`

**为什么重要**：很多创始人的创业动机可以追溯到童年经历。比如一个在祖母厨房里长大的人做了食品品牌，一个小时候经历过健康问题的人做了健康食品——这些故事是文章的灵魂。

### 三、成长经历与人生转折（重点搜索）⭐ 新增重点

- 从青少年到创业前，经历了哪些重要的人生阶段？
- 有没有改变人生方向的关键转折点？（辍学、转行、失业、旅居海外、生病、遇到某个人等）
- 有没有失败经历？怎么度过的？
- 有没有"顿悟时刻"——突然看清了某件事、决定了要做某件事？
- 在不同阶段受到了谁的影响？（导师、合伙人、家人、偶像）

**搜索关键词**：
1. `{人名} journey / story / path`
2. `{人名} turning point / pivotal moment / epiphany`
3. `{人名} before founding / before starting`
4. `{人名} 经历 / 故事 / 历程`

### 四、教育背景（重点搜索）

- 中小学（如有公开信息，包括学校类型、是否转学）
- 本科院校、专业、毕业年份
- 研究生/MBA院校和专业
- 是否有其他培训、认证、进修经历
- 在校期间有没有特别的经历？（社团、实习、竞赛、交换、创业项目）
- 为什么选择这个专业？专业背景对后续创业的影响

**为什么重要**：教育背景往往直接解释了创业方向。比如Pure软糖的关键人物礒辺友里子是设计相关专业出身，这直接影响了她对产品外观和体验的执着。

### 五、职业经历（重点搜索）

- 创业前在哪些公司工作过？
- 担任过什么职位？
- 在每个岗位上做了什么？学到了什么？
- 有没有跨行业的经历？
- 哪段工作经历对后来的创业影响最大？
- 有没有在某个岗位上遇到了重大的挑战或成就？

**搜索渠道**：
1. LinkedIn个人主页（最重要的履历来源）
2. 公司官网的管理层介绍页
3. 行业会议的演讲者介绍
4. 采访文章中的背景介绍段落
5. 商业数据库（如Crunchbase、Bloomberg）

### 六、个人喜好与生活方式（重点搜索）⭐ 新增重点

- 兴趣爱好是什么？（运动、阅读、音乐、旅行、烹饪等）
- 饮食习惯和态度（对食品饮料品牌创始人极其重要——他/她本人是否是品牌理念的践行者？）
- 生活方式（早起型？工作狂？注重工作生活平衡？）
- 推崇的理念或价值观
- 经常引用的话、推荐的书、崇拜的人物
- 个人的标志性特征（如总是穿某种风格的衣服、有某种口头禅）

**搜索关键词**：
1. `{人名} hobbies / interests / passion`
2. `{人名} daily routine / lifestyle / diet`
3. `{人名} favorite book / inspiration / philosophy`
4. `{人名} 爱好 / 日常 / 生活方式`

**为什么重要**：读者对真实的人的真实生活细节有天然的好奇心。一个每天凌晨4点起床跑步的创始人，和一个喜欢在厨房里实验新口味的创始人，给读者的印象完全不同。这些细节让人物变得立体。

### 七、重大人生经历（重点搜索）⭐ 新增重点

- 有没有经历过重大的健康问题？（自己或家人）
- 有没有经历过重大的财务危机？
- 有没有经历过文化冲击或身份认同的挣扎？（如移民、跨文化生活）
- 有没有经历过影响世界观的重大事件？（如金融危机、疫情、自然灾害）
- 这些经历如何塑造了他/她的创业理念和品牌价值观？

**搜索关键词**：
1. `{人名} challenge / struggle / overcome`
2. `{人名} health / crisis / setback`
3. `{人名} life changing / transformative`
4. `{人名} 挑战 / 困难 / 克服`

### 八、创业/加入品牌的背景（有料就写）

- 什么契机让他/她创立/加入这个品牌？
- 之前的经历如何为这件事做了铺垫？
- 有没有个人故事或触动？（如健康问题、家庭原因、旅行中的发现等）
- 创业初期最大的困难是什么？怎么解决的？

### 九、管理/领导风格（有料就写）

- 公开采访中透露的管理理念
- 决策风格（数据驱动？直觉型？共识型？）
- 对创新的态度
- 对失败的态度

### 十、社交媒体与公开形象（有料就写）

- LinkedIn活跃程度及内容倾向
- Twitter/X、Instagram等社交平台的公开信息
- 公开演讲的风格和频率
- 是否经常接受媒体采访

### 十一、家庭背景与个人花絮（有料就写，仅限公开信息）

- 父母的职业和背景（如公开）
- 配偶/伴侣的背景（如公开且与品牌相关）
- 是否有子女？子女是否影响了创业决策？（如很多健康食品创始人是因为有了孩子才关注食品安全）
- 其他有趣的个人花絮

## 搜索策略

### 第一轮：LinkedIn（最重要）

搜索关键词：
1. `{人名} LinkedIn`
2. `{人名} {公司名} LinkedIn`
3. 直接访问LinkedIn个人主页URL（如果能从搜索结果中找到）

**LinkedIn信息的价值**：
- 完整的职业时间线
- 教育背景
- 技能标签
- 自我介绍
- 曾经任职的公司

### 第二轮：官方渠道

1. `{公司名} leadership / management / team`（公司官网管理层页面）
2. `{公司名} about / our story`（关于我们页面中的人物介绍）
3. `{人名} bio / biography`

### 第三轮：人物深度故事（⭐ 新增重点轮次）

**这一轮专门搜索个人经历类内容，是本模块的核心搜索。**

1. `{人名} childhood / grew up / growing up`
2. `{人名} early life / young / family`
3. `{人名} parents / hometown / origin story`
4. `{人名} journey / story / path / how it started`
5. `{人名} turning point / pivotal moment / life changing`
6. `{人名} hobbies / interests / passion / daily routine`
7. `{人名} challenge / struggle / overcome / failure`
8. `{人名} 小时候 / 童年 / 成长经历 / 人生故事`
9. `{人名} 创业故事 / 为什么创业`

**抓取要求**：找到任何包含个人经历描述的文章，必须用 `web_fetch` 获取全文，**不能只看搜索摘要**。这些文章中往往只有一两段提到了童年或成长经历，但这一两段可能就是整篇文章中最有价值的素材。所以必须抓全文，然后从中提取所有与人物个人经历相关的段落，**原文完整保留**。

### 第四轮：媒体与会议

1. `{人名} interview`
2. `{人名} speaker / keynote / conference`
3. `{人名} {行业关键词}`（如 `{name} food industry` / `{name} beverage`）
4. `{人名} podcast`
5. `{人名} TED / TEDx`
6. `{人名} profile / feature / portrait`（人物特写类文章最容易包含个人经历）

### 第五轮：商业数据库

1. `{人名} Crunchbase`
2. `{人名} Bloomberg`
3. `{人名} Forbes`
4. `{公司名} founders Crunchbase`

### 第六轮：社交平台

1. `{人名} Twitter` / `{人名} X.com`
2. `{人名} Instagram`
3. `{人名} Facebook`（公开主页）

### 第七轮：补充搜索——挖掘遗漏的个人经历（⭐ 新增）

在前六轮搜索完成后，回顾已有素材，检查以下维度是否有缺口：

| 维度 | 是否已有素材？ | 如缺失，尝试搜索 |
|------|-------------|----------------|
| 童年/成长环境 | ☐ | `{人名} 童年` / `{人名} grew up` |
| 家庭背景 | ☐ | `{人名} parents` / `{人名} family background` |
| 教育经历细节 | ☐ | `{人名} university` / `{人名} school` |
| 个人喜好 | ☐ | `{人名} favorite` / `{人名} loves` |
| 重大人生转折 | ☐ | `{人名} changed my life` / `{人名} 转折` |
| 健康/个人触动 | ☐ | `{人名} health` / `{人名} personal story` |
| 创业前的失败/弯路 | ☐ | `{人名} before {品牌名}` / `{人名} first job` |

**每个缺失维度至少再搜索1-2次。**

## 来源处理要求

1. **LinkedIn页面**：完整记录所有可见的职业经历和教育信息。因为LinkedIn页面内容可能随时变化，所以要完整记录当前可见的所有信息。
2. **官网管理层页面**：完整保留原文介绍+中文翻译。
3. **演讲者简介**：完整保留原文+中文翻译。这些简介通常非常精炼，每个字都有信息量。
4. **采访中的背景段落**：提取与人物背景相关的段落，完整保留+翻译。
5. **⭐ 人物特写/深度采访中的个人经历段落**：这是最核心的素材来源。任何提到创始人童年、成长、家庭、喜好、重大经历的段落，**必须原文一字不漏地保留**，然后做完整中文翻译。不要概括，不要缩写，不要只提取"关键信息"。原文全文保留是铁律。
6. **播客/视频中的个人经历片段**：如果创始人在播客或视频中讲述了个人故事，必须获取完整transcript/字幕，并保留所有与个人经历相关的段落原文。

### ⚠️ 特别警告：不要遗漏分散在长文中的个人经历片段

很多采访文章长达几千字，其中关于创始人个人经历的内容可能只有2-3段，散落在文章的不同位置。**你必须通读全文，找出所有这些段落，一段都不能漏。**

常见的遗漏场景：
- 文章开头有一段创始人小时候的故事，但你只关注了中间的商业内容
- 文章结尾有一段创始人谈个人喜好，但你在中间就停止了提取
- 问答式采访中，某个问题引出了一段意外的个人经历分享
- 侧边栏或引用框中有创始人的个人趣事

**解决方案**：抓取全文后，用以下关键词在原文中搜索，确保没有遗漏：
- childhood / grew up / young / kid / child / school / college / university
- parents / mother / father / family / brother / sister
- hobby / passion / love / favorite / enjoy
- challenge / difficult / struggle / failure / setback / health
- 小时候 / 童年 / 成长 / 父母 / 家庭 / 爱好 / 困难

## 调研自检

1. 是否找到了所有关键人物（创始人 + CEO + 关键推动者）？
2. 每个人物是否都搜索了LinkedIn？
3. 教育背景和职业经历是否尽可能完整？
4. ⭐ **童年与早期经历**：是否搜索了至少3个不同关键词组合？找到的内容是否已全文保留？
5. ⭐ **成长经历与人生转折**：是否搜索了"journey/story/turning point"等关键词？
6. ⭐ **个人喜好与生活方式**：是否搜索了"hobbies/interests/daily routine"等关键词？
7. ⭐ **重大人生经历**：是否搜索了"challenge/struggle/health"等关键词？
8. ⭐ **原文完整性**：所有涉及个人经历的原文段落是否都已完整保留在 `sources/` 中？是否有被概括或省略的情况？
9. 是否保留了原文+中文翻译？
10. 每条信息是否都有来源？
11. 没有信息的部分是否标注了"未找到公开信息"？
12. 是否避免了编造和臆测？
13. ⭐ 是否回顾了所有已抓取的长文，确认没有遗漏散落在文章各处的个人经历片段？
